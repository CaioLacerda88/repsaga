Phase 38d — Age capture (birth-year)

Optional birth-year so cardio scores on real age-decade norms instead of the age-35 fallback · brand violet #A78BFA / hotViolet
Reuses the row + bottom-sheet grammar of GenderRow / BodyweightRow (label · dimmed value · chevron → sheet with title + helper + control + Cancel/Save). Unlike gender/bodyweight (LGPD Art. 11 sensitive → Hive consent toggle), DOB is Art. 6 consent like avatars: a pure point-of-collection disclosure, no consent toggle. The control is a branded birth-year wheel — NOT a Material calendar (a calendar over-collects month/day; we only key on age-decade). The ≥18 floor is structural: the wheel simply cannot scroll past currentYear−18, so it never re-asks the signup 18+ gate. Default rest = currentYear−35, so skip == fallback. Three surfaces below; en + pt-BR copy throughout.
