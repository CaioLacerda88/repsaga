Phase 38d — Age capture (birth-year)

Optional birth-year so cardio scores on real age-decade norms instead of the age-35 fallback · brand violet #A78BFA / hotViolet
Reuses the row + bottom-sheet grammar of GenderRow / BodyweightRow (label · dimmed value · chevron → sheet with title + helper + control + Cancel/Save). Unlike gender/bodyweight (LGPD Art. 11 sensitive → Hive consent toggle), DOB is Art. 6 consent like avatars: a pure point-of-collection disclosure, no consent toggle. The control is a branded birth-year wheel — NOT a Material calendar (a calendar over-collects month/day; we only key on age-decade). The ≥18 floor is structural: the wheel simply cannot scroll past currentYear−18, so it never re-asks the signup 18+ gate. Default rest = currentYear−35, so skip == fallback. Three surfaces below; en + pt-BR copy throughout.

Notes for build (tech-lead) — 38d:
• Row: clone GenderRow (id profile-age-row, placed after the gender section). Subtitle = derived age string ("39 anos" / "39") via AppTextStyles.numeric-register when set, onSurface@0.55 "Não informado" / "Not set" when null. No warning icon.
• Sheet: showModalBottomSheet(isScrollControlled:true); title AppTextStyles.title.copyWith(fontSize:20); one-line disclosure helper (body @ onSurface@0.65) — point-of-collection only, NO consent banner/toggle (DOB = LGPD Art. 6, not Art. 11). Save → upsertProfile(dateOfBirth: DateTime(year,1,1)) → ref.invalidate(profileProvider) → pop.
• Wheel: ListWheelScrollView.useDelegate, years currentYear−18 … currentYear−100 (top→bottom), initial item = currentYear−35. Center numeral AppTextStyles.numeric in textCream; flanking items dim via onSurface alpha ramp; selection band = hotViolet@0.10 fill + hotViolet@0.45 top/bottom hairlines. itemExtent derived from MediaQuery.textScalerOf(context).scale(...) so it can't clip at large text; cap visible items 5→3 under heavy scale.
• ≥18 floor is structural — the delegate's min child IS currentYear−18; there is no under-18 row to scroll to, so the signup 18+ gate is never re-asked.
• Prefer not to say → writes date_of_birth = null (clears a previously-set value) and pops. Cancel / Save row = dialogTextButtonStyle / dialogFilledButtonStyle (gender-sheet parity).
• Post-session prompt: one-shot in post_session_screen.dart (analog of _fireMountAnalytics); gate to bp == BodyPart.cardio sessions with null DOB. Dismissal via a Notifier<bool> over Hive userPrefs (clone bodyweight_prompt_dismissal_provider); "Set age" opens the same AgeEditorSheet.
• No Material showDatePicker, no calendar, no month/day collection, no gradient, no heroGold (reward-scarcity).
