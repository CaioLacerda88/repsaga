Semântica Dourado = recorde atual (campeão histórico). Não é "isto foi um PR no momento da conclusão" — é "isto é o melhor que você já fez neste exercício, agora". Por isso só uma linha de cada exercício pode ter a moldura dourada por vez. Quando uma série posterior supera, a anterior cai para o estado "PR superado".
Regra Dourado aparece em somente três lugares na linha de PR atual: barra esquerda, valor do PR, moldura direita do done-col. Botões +/-, número da série, unidade ("reps", "kg") nunca recebem dourado — heroGold é reservado para conquista, nunca para affordance interativo.
Alinhamento Valor PR mantém mesma fonte-size, mesma altura de linha, mesma baseline que valores normais. A unidade ("reps") é literal em todas as linhas. Linhas têm altura idêntica independente do estado.
